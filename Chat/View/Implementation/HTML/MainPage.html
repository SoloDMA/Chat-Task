Welcome to chat!

В нашем чате есть боты. Чтобы обратиться к какому-либо боту используйте следующий формат сообщения:"@[имя_бота] /[команда_бота] [аргумент_команды (опционально)]"

Список ботов:

1. joke - присылает анекдот по команде /joke

2. time - присылает текущее время по команде /current

3. downloader - присылает содержимое тега title по команде /site (имеет аргумент команды - адрес сайта)
